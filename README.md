# TrabalheNaTegraDesafio
Solução: sorteio de desenvolvedores para realizar pair programming.

# Dependências necessárias : Nodejs ,npm , bower, gulp

# Rodando o projeto: 

    - $ npm install
    - $ bower install
    - $ gulp
    - $ gulp watch
